# Kirill Kushov

**Fullstack / Backend Developer**  
Python · FastAPI · PostgreSQL · React · TypeScript · Docker

I build production-oriented web applications with a focus on clean backend architecture, reliable APIs, database design, and practical user-facing interfaces.

My main strength is turning product requirements into working systems: authentication, role-based access, admin panels, dashboards, import/export flows, background jobs, and Dockerized environments.

## Core Expertise

- Backend architecture with **Python, FastAPI, SQLAlchemy, Alembic**
- REST API design, authentication, JWT, RBAC and multi-role systems
- PostgreSQL schema design, migrations, relationships and query optimization basics
- Frontend development with **React, TypeScript, Vite**
- Admin dashboards, CRUD interfaces, forms, tables and data workflows
- Docker / Docker Compose environments for local and production-like development
- CSV/XLSX import-export pipelines
- Git, GitHub, repository hygiene and delivery-focused development
